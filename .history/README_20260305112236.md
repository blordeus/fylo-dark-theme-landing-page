# Fylo Dark Theme Landing Page

A responsive landing page built from the Frontend Mentor “Fylo dark theme landing page” challenge. Focus areas: layout, responsive behavior, mobile navigation, and email form validation.

## Links
- Live site: https://blordeus.github.io/fylo-dark-theme-landing-page/
- Challenge: https://www.frontendmentor.io/challenges/fylo-dark-theme-landing-page-5ca5f2d21e82137ec91a50fd

## Features
- Responsive layout (mobile, tablet, desktop)
- Mobile hamburger menu with overlay + Escape-to-close
- Email validation with inline error messaging
- Hover/focus states for interactive elements

## Built With
- Semantic HTML5
- CSS3 (custom)
- Vanilla JavaScript
